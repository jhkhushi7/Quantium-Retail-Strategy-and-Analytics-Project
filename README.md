# Quantium-Retail-Strategy-and-Analytics-Project
This project is a part of **Quantium's Virtual Internship**. It focuses on **retail data analytics** - analyzing customer purchasing behaviour and deriving insights to help a supermarket client improve it's strategy.

## Project Overview
Using Python and analytical thinking, this project covers :
- cleaning and prepparing transactional and customer datasets
- Performing **Exploratory Data Analysis(EDA)** using `pandas`,`matplotlob` and `seaborn`
- Creating and derived features like **pack size** and **brand** from product names
- Merging transaction and customer data for detailed insights
- Visualizing **sales trends** across different customer segments
-  Conducting **statistical testing and A/B testing** to validate insights 
- Providing actionable recommendations for marketing and product strategies  

---

## Key Insights
- **Mainstream Young Singles/Couples** and **Older Families** were the top contributors to chip sales.  
- **Mainstream customers** showed higher average spending per unit, preferring slightly premium brands.  
- Buying behavior and pack size preferences varied across lifestages and customer types, indicating opportunities for **targeted promotions**.
